# 📝 Registration Form Webpage

## 📌 Project Overview

This project demonstrates the creation of a **basic HTML Registration Form** that collects user information such as name, email, password, account type, age, and profile details.

The webpage is designed using fundamental HTML form elements to allow users to provide personal information and submit it through a structured form.

This project highlights important HTML concepts including **form elements, input types, radio buttons, dropdown menus, file upload, and text areas**.

---

## 🎯 Problem Statement

To design a user registration form that collects essential user details using HTML form elements and organizes them in a structured and user-friendly format.

---

## 🛠️ Project Modules Covered

### 1️⃣ HTML Document Structure
- Created the basic webpage structure using `html`, `head`, and `body`.
- Included meta tags for character encoding and responsive design.

### 2️⃣ Header Section
- Added a page title using `<h1>`.
- Provided instructions for users to fill out the form.

### 3️⃣ Text Input Fields
- Collected user details such as:
  - First Name
  - Last Name
  - Email
  - Password

### 4️⃣ Radio Buttons
- Allowed users to select their **account type**:
  - Personal
  - Business

### 5️⃣ File Upload
- Enabled users to upload a **profile picture**.

### 6️⃣ Numeric Input
- Collected user's **age** using number input.

### 7️⃣ Dropdown Selection
- Provided a **select menu** to ask how users heard about the platform.

### 8️⃣ Text Area
- Allowed users to write a **short bio**.

### 9️⃣ Terms and Conditions
- Added a checkbox to accept **Terms and Conditions**.

### 🔟 Form Submission
- Used a submit button to send form data.

### 1️⃣1️⃣ Footer Section
- Added contact information and useful links.
